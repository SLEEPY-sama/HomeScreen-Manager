# 🎮 HomeScreenManager

A lightweight and reusable Unity C# script for managing game home screens and menu navigation.

This project was created during my early Unity learning journey as a way to understand UI management, scene loading, and button interactions. It now serves as a reusable template that can be integrated into future Unity projects whenever a simple main menu system is needed.

---

## 📌 Overview

`HomeScreenManager.cs` is a simple menu controller designed to handle common home screen functionality in Unity games.

It provides a centralized script for menu-related actions, making it easier to organize UI behavior and reduce repetitive setup across projects.

---

## ✨ Features

* ▶️ Start Game Button
* ⚙️ Settings Navigation
* 📖 Instructions / Help Screen Navigation
* 🚪 Quit Game Functionality
* 🔄 Scene Management Support
* 🎯 Easy Button Integration
* ♻️ Reusable Across Multiple Projects

---

## 🛠️ Built With

* Unity Engine
* C#
* Unity UI System
* SceneManager API


---

## 🚀 Usage

1. Import the script into your Unity project.
2. Attach `HomeScreenManager.cs` to a GameObject in your menu scene.
3. Connect UI Buttons through the Unity Inspector.
4. Assign scene names where required.
5. Run the game and test menu navigation.
